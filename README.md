# Installation 
1.- Download the latest zip file from releases<br/>
2.- Extract the Pokemon_Little_Little_Village dir <br/>
3.- Launch the .exe file inside

# Advanced Installation (Requires Python)

```

cd Pokemon_Little_Little_Village
python3 main.py
```
